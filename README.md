新版 Stack 的支持插件，基于 PaperMC。

## 安装

你可以通过构建本项目取得 `jar` 文件并安装。

### 依赖

* Paper 服务器
* CommandAPI

普通 Spigot 不行。Paper 的各种 Fork 也许可以运行，但我们不保证能不能运行。
